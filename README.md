# Liberty Insurance Mockup

Static homepage concept for Liberty Insurance Agency.

## Preview

Open `index.html` in a browser, or run:

```bash
python3 -m http.server 8000
```

Then visit `http://127.0.0.1:8000`.

## Concept

The mockup reframes Liberty as a strategic risk advisor with:

- Business, employee benefits, and personal protection paths
- Interactive industry-specific risk panels
- A visual Liberty Advantage process
- A client command center concept for training, compliance, and claims readiness
